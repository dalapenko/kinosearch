# KinoSearch App [Android]

## Описание

Выпускной проект по Android курсу от Otus.

Приложения просмотра текущих премьер в кино и релизов на цифровых носителях.

## Использованный стек

- Single Activity
- MVVM
- Navigation Component
- Coroutine/Flow
- Hilt
- Retrofit
- Room
- Kotlin Serialization
